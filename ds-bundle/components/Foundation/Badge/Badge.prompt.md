# Badge

Accent-blue pill label. Uppercase text with subtle background. Always sits above a title.

## Usage

```tsx
<Badge>WEEK 3 · DAY 2</Badge>
<Badge>PUSH DAY</Badge>
<Badge>REST DAY</Badge>
```

## Rules
- Text is uppercased automatically via CSS — pass lowercase if preferred.
- Use only for contextual labels (week/day/phase info). Not for status or counts.
- One badge per header maximum.
